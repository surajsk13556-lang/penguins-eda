# Palmer Penguins Dataset - Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on the Palmer Penguins dataset to understand the characteristics of different penguin species and identify patterns in their physical measurements.

## Project Overview

The Palmer Penguins dataset is a popular alternative to the Titanic dataset for learning data analysis and visualization techniques. This project explores penguin species, body measurements, and relationships between different variables.

## Dataset Features

The dataset contains information about penguins, including:

* Species
* Island
* Bill Length (mm)
* Bill Depth (mm)
* Flipper Length (mm)
* Body Mass (g)
* Sex

## Key Findings

1. Different penguin species show distinct physical characteristics.
2. Body mass is related to flipper length and bill measurements.
3. Correlation analysis helps identify relationships among numerical features.
4. Data visualization makes species-wise comparison easier.

## Conclusion

This Exploratory Data Analysis provides insights into penguin species characteristics and demonstrates fundamental data analysis techniques such as data cleaning, visualization, and correlation analysis.
